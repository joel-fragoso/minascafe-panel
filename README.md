### Como clonar o projeto:
```bash
# SSH
$ git clone git@github.com:joel-fragoso/minascafe-panel.git

# HTTPS
$ git clone https://github.com/joel-fragoso/minascafe-panel.git
```

### Como instalar as dependências:
```bash
$ yarn install
```

### Como executar a aplicação:
```bash
$ yarn dev
```
